This project focuses on building a Retail Sales Data Agent using Databricks to analyze retail transaction data and generate accurate, data-driven business insights. The agent evaluates sales performance, customer behavior, product trends, and profitability while answering questions strictly based on the provided dataset without fabricating information. Using tools such as SQL and Python, the project demonstrates how AI-powered analytics can support smarter business decisions through trend analysis, forecasting, customer segmentation, and actionable recommendations.
